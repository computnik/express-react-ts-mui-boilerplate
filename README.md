# express-react-ts-mui-boilerplate
> Computnik's Fullstack Boilerplate

This is a fullstack boilerplate with features like:
1. React Frontend
2. Express Backend
3. Material UI
4. Webpack
5. TypeScript

## Usage

```shell
npx degit computnik/boilerplate
```
